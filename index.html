2.创建后缀名为.cpp的源文件

指针：
可以通过指针间接访问内存
//定义指针
int a = 10;
//指针定义的语法： 数据类型 * 指针变量名
int * p;
//让指针记录变量a的地址
p = &a;
//指针前面加*代表解引用，找到指针指向的内存中的数据
*p == a;

空指针：
指向内存中编号为0的空间
用途：初始化指针变量（如一开始不知道指针指向哪好的时候就指向编号为0的空间）
注意：空指针指向的内存是不可以访问的
int * p = NULL;

野指针：
int * p = (int )0x1100; //指针变量p指向内存地址编号为0x1100的空间
cout<<p << endl; //访问野指针报错

结论：空指针和野指针都不是我们申请的空间，所以不要随便的访问他们

const int * p = &a;(常量指针)：
特点：指针的指向可以修改，但是指指向的值不可以改
*p = 20; //错误
p = &b; //正确

int * const p = &a;(指针常量)：
特点：指针的指向不可以改，指针指向的值可以改
*p = 20; //正确
p = &b; //错误

const int * const p = &a;(即修饰指针，又修饰常量)：
特点：指针的指向和指针指向的值都不可以改
*p = 20; //错误
p = &b; //错误

另外的，访问数组名就是访问数组的首地址

值传递与地址传递：
值传递只变化形参，地址传递可以修饰实参

.h结尾的是头文件，当中放函数的声明；.cpp结尾的文件是源文件，放主函数和函数的定义

举个例子：用冒泡排序排列一个数组
主函数：
int main() {
int arr[10] = { 4,3,6,9,1,2,10,8,7,5 }; //创建一个数组
int len = sizeof(arr) / sizeof(arr[0]); //求出数组的长度
bubbleSort(arr, len); //调用bubbleSort函数
printArray(arr, len);
return 0;
}
冒泡排序函数：
void bubbleSort(int* arr, int len)
{
for (int i = 0; i < len - 1; i++)
{
for(int j = 0; j < len - i - 1; j++)
if (arr[j] > arr[j + 1])
{
int temp = arr[j];
arr[j] = arr[j + 1];
arr[j + 1] = temp;
}
}
}
打印函数：
void printArray(int* arr, int len)
{
for (int i = 0; i < len; i++)
{
cout << arr[i] << endl;
}
}

结构体：
结构体属于用户自定义的数据类型，允许用户储存不同的数据类型。
语法：struct 结构体名 {结构体成员列表};
如何使用结构体呢？ 举个例子：

  2. struct Student s2 = {….};
  3. 再定义结构体的时候顺便创建结构体变量；
    访问方法：stuArray.name
    stuArray.age
....
    //创建结构体数组
  4. struct Student stuArray[3]=
    {
    {“张三”, 18, 100},
    {“李四”, 28, 99},
    {“王五”, 38, 66}
    };
  5. stuArray[2].name = “肥龙”;
    ….
    访问方法：stuArray[i].name
    stuArray[i].age

结构体指针：
通过指针来访问结构体中的成员
利用操作符->访问结构体属性
student s = {“张三”, 18, 100}; //创建学生结构体变量
student * p = &s; //通过指针指向结构体变量
p->name; //通过指针访问结构体变量中的数据

结构体函数参数:
值传递：
printStudent(s);
void printStudent(struct student s)
地址传递：
printStudent(&s);
void printStudent(struct student * p)

将函数中的形参改为指针，可以减少使用的内存空间，而且不会复制一个新的副本出来
void printStudent(const student * stu) //const用来防止误操作

c++补充知识1.0

今天是3月15号。
肥龙他补充了一天的c++的知识。
包括但不限于以下这些知识。
函数，结构体等不出意外将在下一篇博客记录。

变量的存在意义：
方便我们管理内存空间

C++定义常量：
1.#define: #define 常量名 常量值 如：#define a 10;
2.const 数据类型 常量名 常量值 如:const int month = 12;

关键字:
关键字是C++中预先保留的单词如int char等等

标识符命名规则（即给变量取名的规则）：
1.不能为关键字
2.标识符只能由数字，字母，下划线组成
3.第一个字符必须为字母或者下划线
4.区分大小写

sizeof 关键字：
利用sizeof关键字可以统计数据类型所占内存大小。 sizeof(数据类型/变量)
如： cout<< “short占用内存空间为：” <<sizeof(short)<<endl;

ASCII码：
简单理解为用二进制数来表示一些特定的字符。

转义字符：
用来表示一些不能显示出来的ASCII字符
常见的有： \n 换行，\t 水平制表TAB（补齐8位），\ 表示一个反斜杠

字符串型：
1.c风格： char 变量名[] = “字符串值”
2.c++风格: string 变量名 = “字符串值”

bool类型：
true = 1;
false = 0;
//bool flag = true;
cout << flag << endl;(输出为1)

算术运算符：
% 取模（取余数） 如： 10 % 3 = 1;
前置递增：++a ；先自身加一，然后在运算其他的东西；
后置递增：a++ ；先运算其他的表达式，然后再自身加一；
递减同理；

赋值运算符：
+=: a=0;a+=2; 结果：a=2;
-=: a=5;a-=2; 结果：a=3;
*=和=和%=同理；

逻辑运算：
&&：与
||：或

三目运算符：
语法：表达式1？表达式2：表达式3；
c = ( a>b ? a : b); //返回的值是变量，可以继续赋值。

添加随机数种子：
#include //头文件
srand((unsigned int)time(NULL)); //随机种子
int num = rand() % 100 + 1; 生成0+1到99+1中的随机数

跳转语句：
break：退出
continue：跳过本次循环中余下尚未执行的语句，继续执行循环
goto：无条件跳转语句 //goto 标记

这是一只肥龙？

今天是20年的3月12日呢，是我第一次和这只肥龙见面。他并没有照片中那么丑额！

可是这一身上的颓势是怎么回事呢？

从肥龙的口中说出了如下一段话：
那场失败也许是刻在了我的脑海里了
那是一年多前的一次比赛了
是的，就是一年多前
那时，“想改变自己，改变下这糜烂的生活”的想法已经萌发了
努力力，那个名次，和自己讲还没那么糟糕，生活好起来了？？？？？
是一场数学建模的竞赛
这里就说一下结果：是毫无疑问的大败
过程这里不想细说，也许之后的博文会提到。。
那时的队员加上自己是3个，并没有这方面的大佬，队员也是自己找的。
自这场竞赛之后，即使有想萌发再去参加比赛的念头，也没有真正的行动去报名了

可以说是续那场竞赛之后，第一个team
三个人 和一个共同实现的目标
我有反问过自己，自己可以成为别人的依靠了吗？靠谱了吗？
答案很显然：并没有
但也许这次可以使我真正的成长了
三个不靠谱的男人的结果会是什么样的呢？
。。。。。。。。。

直面自己的内心，都是他夜里躺着床上睡前才会做的事情。睡一觉后起个床后，负心汉就会完全忘记自己昨晚对自己定下的目标[○･｀Д´･ ○]

但我知道这次的课程设计对于他的重要性，不管是哪个方面的
3个月的时间来揭晓谜底

如果成功了又会是怎么的呢？
