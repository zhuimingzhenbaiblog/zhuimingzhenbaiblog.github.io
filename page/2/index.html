静态成员变量：
所有对象共享一份数据
在编译阶段分配内存
类内声明，内外初始化
比如在某个类A中声明一个static int number；初始化为0。这个number就能被所有A的实例共用。在A的构造函数里加上number++，在A的析构函数里加上number–。那么每生成一个A的实例，number就加一，每销毁一个A的实例，number就减一，这样，number就可以记录程序中共生成了多少个A的实例。

静态成员函数：
所有对象共享同一个函数
静态成员函数只能访问静态成员变量

{
public:

{
……;
}
static int m_A; //静态成员变量在类内声明

int Person::m_A = 0; //在类外初始化
访问方式：
1.通过对象访问：
Person p;
p.func();
2.通过类名访问：

c++补充知识3.0

int a 和 int &a 的区别：
int a的意思是定义一个变量a
int &a意思是定义一个引用 //引用相当于指针再取值 他和被引用的变量都是表示同一块内存
引用就是给变量取别名 int b ;int &a=b; 用cout<<&a 和cout<<&b 你会发现他们的地址是一样的

C++核心编程：
内存分区模型：
#代码区：存放函数体的二进制代码，有操作系统进行管理。（就是自己写的代码的内容就是放在代码区中）
#全局区：存放全局变量和静态变量以及常量。
#栈区：由编译器自动分配释放，存放函数的参数值，局部变量等。（该地区的数据存活死亡由编译器管理）
#堆区:由程序员分配和释放，若程序员不释放，程序结束时由操作系统回收。
内存分区的意义：
给数据不同的生命周期啊，内存管理会更加方便，合理，反正大大的好。

拷贝构造函数的调用时机:
    1.使用一个已经创建完毕的对象来初始化一个新对象
    2.值传递的方式给函数参数传值
    3.以值方式返回局部变量

指针：
可以通过指针间接访问内存
//定义指针
int a = 10;
//指针定义的语法： 数据类型 * 指针变量名
int * p;
//让指针记录变量a的地址
p = &a;
//指针前面加*代表解引用，找到指针指向的内存中的数据
*p == a;

空指针：
指向内存中编号为0的空间
用途：初始化指针变量（如一开始不知道指针指向哪好的时候就指向编号为0的空间）
注意：空指针指向的内存是不可以访问的
int * p = NULL;

野指针：
int * p = (int )0x1100; //指针变量p指向内存地址编号为0x1100的空间
cout<<p << endl; //访问野指针报错

结论：空指针和野指针都不是我们申请的空间，所以不要随便的访问他们

const int * p = &a;(常量指针)：
特点：指针的指向可以修改，但是指指向的值不可以改
*p = 20; //错误
p = &b; //正确

int * const p = &a;(指针常量)：
特点：指针的指向不可以改，指针指向的值可以改
*p = 20; //正确
p = &b; //错误

const int * const p = &a;(即修饰指针，又修饰常量)：
特点：指针的指向和指针指向的值都不可以改
*p = 20; //错误
p = &b; //错误

另外的，访问数组名就是访问数组的首地址

值传递与地址传递：
值传递只变化形参，地址传递可以修饰实参

    访问方法：stuArray.name
    stuArray.age
....
    //创建结构体数组

结构体指针：
通过指针来访问结构体中的成员

c++补充知识1.0

今天是3月15号。
肥龙他补充了一天的c++的知识。
包括但不限于以下这些知识。
函数，结构体等不出意外将在下一篇博客记录。

变量的存在意义：
方便我们管理内存空间

C++定义常量：
1.#define: #define 常量名 常量值 如：#define a 10;
2.const 数据类型 常量名 常量值 如:const int month = 12;

关键字:
关键字是C++中预先保留的单词如int char等等

标识符命名规则（即给变量取名的规则）：
1.不能为关键字
2.标识符只能由数字，字母，下划线组成
3.第一个字符必须为字母或者下划线
4.区分大小写

sizeof 关键字：
利用sizeof关键字可以统计数据类型所占内存大小。 sizeof(数据类型/变量)
如： cout<< “short占用内存空间为：” <<sizeof(short)<<endl;

ASCII码：
简单理解为用二进制数来表示一些特定的字符。

转义字符：
用来表示一些不能显示出来的ASCII字符
常见的有： \n 换行，\t 水平制表TAB（补齐8位），\ 表示一个反斜杠

字符串型：
1.c风格： char 变量名[] = “字符串值”
2.c++风格: string 变量名 = “字符串值”

bool类型：
true = 1;
false = 0;
//bool flag = true;
cout << flag << endl;(输出为1)

算术运算符：
% 取模（取余数） 如： 10 % 3 = 1;
前置递增：++a ；先自身加一，然后在运算其他的东西；
后置递增：a++ ；先运算其他的表达式，然后再自身加一；
递减同理；

赋值运算符：
+=: a=0;a+=2; 结果：a=2;
-=: a=5;a-=2; 结果：a=3;
*=和=和%=同理；

逻辑运算：
&&：与
||：或

三目运算符：
语法：表达式1？表达式2：表达式3；
c = ( a>b ? a : b); //返回的值是变量，可以继续赋值。

添加随机数种子：
#include //头文件
srand((unsigned int)time(NULL)); //随机种子
int num = rand() % 100 + 1; 生成0+1到99+1中的随机数

跳转语句：
break：退出
continue：跳过本次循环中余下尚未执行的语句，继续执行循环
goto：无条件跳转语句 //goto 标记
