指针：
可以通过指针间接访问内存
//定义指针
int a = 10;
//指针定义的语法： 数据类型 * 指针变量名
int * p;
//让指针记录变量a的地址
p = &a;
//指针前面加*代表解引用，找到指针指向的内存中的数据
*p == a;

空指针：
指向内存中编号为0的空间
用途：初始化指针变量（如一开始不知道指针指向哪好的时候就指向编号为0的空间）
注意：空指针指向的内存是不可以访问的
int * p = NULL;

野指针：
int * p = (int )0x1100; //指针变量p指向内存地址编号为0x1100的空间
cout<<p << endl; //访问野指针报错

结论：空指针和野指针都不是我们申请的空间，所以不要随便的访问他们

const int * p = &a;(常量指针)：
特点：指针的指向可以修改，但是指指向的值不可以改
*p = 20; //错误
p = &b; //正确

int * const p = &a;(指针常量)：
特点：指针的指向不可以改，指针指向的值可以改
*p = 20; //正确
p = &b; //错误

const int * const p = &a;(即修饰指针，又修饰常量)：
特点：指针的指向和指针指向的值都不可以改
*p = 20; //错误
p = &b; //错误

另外的，访问数组名就是访问数组的首地址

结构体：
结构体属于用户自定义的数据类型，允许用户储存不同的数据类型。
语法：struct 结构体名 {结构体成员列表};
如何使用结构体呢？ 举个例子：

//创建具体的学生有如下3个方法

  1. struct Student s1; //s1.name = “张三”；s1.age = 18;….
  2. struct Student s2 = {….};
  3. 再定义结构体的时候顺便创建结构体变量；
    访问方法：stuArray.name
    stuArray.age
....
    //创建结构体数组
  4. struct Student stuArray[3]=
    {
    {“张三”, 18, 100},
    {“李四”, 28, 99},
    {“王五”, 38, 66}
    };
  5. stuArray[2].name = “肥龙”;
    ….
    访问方法：stuArray[i].name
    stuArray[i].age
....

结构体指针：
通过指针来访问结构体中的成员
利用操作符->访问结构体属性
student s = {“张三”, 18, 100}; //创建学生结构体变量
student * p = &s; //通过指针指向结构体变量
p->name; //通过指针访问结构体变量中的数据

结构体函数参数:
值传递：
printStudent(s);
void printStudent(struct student s)
地址传递：
printStudent(&s);
void printStudent(struct student * p)

将函数中的形参改为指针，可以减少使用的内存空间，而且不会复制一个新的副本出来
void printStudent(const student * stu) //const用来防止误操作
