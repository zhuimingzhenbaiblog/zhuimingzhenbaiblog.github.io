冒泡排序

冒泡排序是一种排序算法

重复的走访要排序的数列，一次比较两个元素
下面是一张动图，表示的很直观，动图来自：https://www.cnblogs.com/onepixel/articles/7674659.html

就想GIF中所展现的那样，越小的元素会慢慢的“浮”到数列的顶端，就像冒泡一样
以下是c++语言来实现该算法：
void bubbleSort(int arr[], int len)
{
for (int i = 0; i < len - 1; i++)
for (int j = 0; j < len - 1 - i; j++)
{
if (arr[j] > arr[j + 1]) //相邻两元素对比
{
int temp = arr[j + 1]; //元素交换
arr[j + 1] = arr[j];
arr[j] = temp;
}
}
}
