c++新增知识2.0

C++中对文件操作要包含头文件
头文件： #include
创建流对象： ofstream ofs;
打开文件： ofs.open(“文件路径”,打开方式);
写数据： ofs << “。。。。”;
关闭文件： ofs.close();
打开方式：
ios::in 为读文件而打开文件
ios::out 为写文件而打开文件
ios::ate 初始位置：文件尾
ios::app 追加方式写文件
ios::trunc 如果文件存在，删除文件然后再创建文件
ios::binary 二进制方式

函数模板：
template
void mySwap(T& a, T& b)
{
T temp = a;
a = b;
b = temp;
}
1.自动推导
mySwap(a, b);
2.显示指定类型
mySwap(a, b);

类模板和函数模板差别：
1.类模板没有自动类型推导的使用方式
2.类模板在模板列表中可以有默认参数

STL（Standard Templata Library 标准模板库）
